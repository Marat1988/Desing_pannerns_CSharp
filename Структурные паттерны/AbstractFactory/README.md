Паттерн "Абстрактная фабрика (Abstract Factory)"

Назначение: абстрактная фабрика предоставляет интерфейс для создания семества взаимосвязанных или родственных объектов (dependent or related objects), не специфицируя их конкретных классов.

Другими словами: абстрактная фабрика предостввляет собой стратегию создания семейства взаимосвязанных или родственных объектов.
